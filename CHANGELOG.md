# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-02-06

### Added

- 添加 Ant Design Vue 4.x UI 组件库
- 添加 Sass 样式预处理器支持
- 添加 Tailwind CSS 4.x 原子化 CSS 框架
- 添加 vite-svg-loader 支持本地 SVG 图标
- 添加环境变量管理（支持多环境配置）
- 添加 Pinia 状态管理和数据持久化
- 添加 Vue Router 4 路由管理
- 添加 Axios API 请求封装
- 添加 vite-plugin-mock 本地 Mock 数据支持
- 添加 vue-i18n 国际化支持

### Changed

- 完善项目工程化配置
- 优化代码结构和组织方式

### Fixed

- 修复代码格式化和类型检查问题

## [1.0.0] - 2026-02-04

### Added

- 初始版本发布
- Vue 3 + TypeScript + Vite 项目模板
- ESLint 9.x 扁平配置
- Prettier 代码格式化
- Stylelint 样式检查
- Husky + lint-staged Git Hooks
- Commitlint 提交信息规范
- EditorConfig 编辑器配置
- VSCode 工作区配置

### Features

- 完整的代码规范体系
- 保存自动格式化
- Git 提交前自动检查
- TypeScript 类型检查
